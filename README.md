# לוֹמְדִים כְּתָב רָשִׁ״י 🔤
### מאפס — עד קריאה שוטפת בכתב רש״י. משחקים, מסע, וארון ספרים שנפתח.

> *"כָּל אֶחָד יָכוֹל לִקְרֹא רָשִׁ״י."*
> היעד: לפתוח דף גמרא, מקראות גדולות או משנה ברורה — ולקרוא בשטף.

---

## ✅ מה זה
אפליקציית לימוד לכתב רש״י, לכל גיל. **אפס עלות, אפס תלות, אפס טוקנים, אפס AI.**
הכול רץ במכשיר: האותיות מרונדרות בפונט רש״י אמיתי (Noto Rashi Hebrew, OFL), ההקראה משתמשת בקול החינמי של הדפדפן, וההתקדמות נשמרת מקומית בלבד.

## 🧩 הרעיון הפדגוגי (מבוסס מחקר)
- **לא לומדים לפי א״ב.** ~18 מ-27 האותיות כמעט זהות לכתב הרגיל — מי שקורא עברית כבר יודע 2/3.
- **מתחילים מהצלחה** — מגלים שכבר מזהים מילה בכתב רש״י.
- **הקושי מרוכז ב-7 אותיות** (א · ב · ד · ז · צ · ץ · ש) + זוג ההיפוך המפורסם **ס/ם**.
- **ניגוד לפני שינון** — אותיות מבלבלות נלמדות תמיד זו־מול־זו.
- **מנוע חזרה מרווחת (SR)** — אות קשה חוזרת תכוף, קלה בהדרגה.
- **מתקדמים לטקסט אמיתי** — קטעי רש״י על התורה, מהקל לקשה.

## 🗺️ מסע 7 העולמות
0. **אתה כבר יודע!** — הוק פתיחה, חשיפת מילה מוכרת.
1. **האותיות הקלות** — 18 האותיות הזהות (התאמה, זיהוי, ארקייד).
2. **הדומות** — זוגות בלבול, תרגילי ניגוד.
3. **הבוס: הקשות** — 7 האותיות + מיני־משחק ה־ם/ס.
4. **מילים** — קריאה והשלמת אותיות.
5. **משפטים ורש״י** — קריאת קטע רש״י אמיתי עם רמזים נעלמים.
6. **שטף** — קריאה חופשית בלי רמזים, עד דף גמרא.

## 🎁 מערכת התגמול (בלי חנות־קניות — מתאים לילדים ולמתגברים)
- **אֲרוֹן הַסְּפָרִים** — כל עולם פותח ספר אמיתי (סידור → חומש → פרקי אבות → מקראות גדולות → עין יעקב → משנה ברורה → דף גמרא), ובכל אחד קוראים קטע אמיתי.
- **חוֹתְמוֹת הַמַּדְפִּיסִים** — אוסף היסטורי (בן־גרטון 1475, שונצינו, בומברג).
- **דרגות לומד** — מתחיל → מפענח → קורא → תלמיד → למדן → בקי → רגיל ברש״י.
- **מדליות · רצף יומי · אתגר יומי · לוח התמדה.**

## 🎭 מתג גיל
- **מצב יֶלֶד** — צבעוני, שובב.
- **מצב בּוֹגֵר** — בית־מדרש (קלף, נייבי, זהב).
מתחלף בכל רגע ב"הישגים → הגדרות".

## 🚀 איך מריצים
**לוקאלי:**
```
cd rashi-app/app
python -m http.server 5330
```
פותחים: http://localhost:5330

**פריסה חינם:** מעלים את תיקיית `app/` ל-Vercel / GitHub Pages. בלי שרת, בלי DB.
**בטלפון:** נכנסים בכרום → "הוסף למסך הבית". עובד אופליין (PWA).

## 📂 מבנה
```
app/
  index.html · manifest.json · sw.js      ← PWA + אופליין
  fonts/  noto-rashi-400/700.woff2 · fonts.css   ← פונט רש״י אמיתי (OFL)
  css/style.css                            ← 2 סקינים (data-mode)
  data/   letters.js · words.js · passages.js · content.js   ← התוכן
  js/     state.js · audio.js · ui.js · games.js · app.js     ← המנוע
docs/  (מחקר פדגוגי + החלטות)
```

## ➕ הוספת תוכן (בלי לגעת במנוע)
- **מילה:** `data/words.js` — `{t:"מנוקד", p:"עיצורים", m:"פירוש", lvl:1-4}`.
- **קטע רש״י:** `data/passages.js` — `{id, lvl, src, t, plain, tr}`.
- **ספר לארון:** `data/content.js` → `SEFORIM`.

## 📖 בֵּית הַמִּדְרָשׁ (חדש)
טאב "בית מדרש" — קריאה בטקסט אמיתי, בעיצוב צורת-הדף:
- **חומש בראשית א׳** — פסוק מנוקד, ואז פירוש רש״י בכתב רש״י (וכן הלאה).
